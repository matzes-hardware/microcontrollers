# Holtek

Eingeschlossen wurden Herstellerbezeichnungen der SOIC-Familie: SOP, NSOP, SSOP sowie EP-Varianten. Ausgeschlossen wurden TSSOP, MSOP, QFN, LQFP, DIP/PDIP und reine Bare-Die/COB-Varianten.

Holtek ist anhand des Holtek Product Guide 2024 sowie einiger direkter Holtek-PDFs erfasst; bei Holtek können einzelne neuere Revisionen Packagelisten geändert haben. Frühere Ausgaben hatten Holtek-Teile teilweise ausgelassen.
